## 공지사항

- 현재 Storybook은 공용 UI 기준으로 정리되어 있으며, 신규 컴포넌트는 동일한 작성 양식을 따라 추가해주세요.
- 작성 기준은 `Docs/Storybook Guide` 문서를 참고해주세요.
- 컴포넌트 API 변경 시, 구현 코드와 Storybook 문서를 같은 PR에서 함께 업데이트하는 것을 원칙으로 합니다.
